# Q-A-App-with-RAG-using-gemini_pro-langchain
AI-powered Q&amp;A app built with Streamlit, LangChain, Gemini Pro, and Chroma vector store. Upload PDFs, process and split them into chunks, generate embeddings, and retrieve contextually relevant answers using RAG and Gemini, all via a conversational UI for document-grounded search.
### Video Tutorial: https://youtu.be/8cKf5GUz4TU

## Environment setup:

		conda create -n env_langchain1 python=3.10  
		conda activate env_langchain1
		python -m pip install --upgrade pip
		Install packages:
		pip install -r requirements.txt


## Run App:
		streamlit run app1.py
